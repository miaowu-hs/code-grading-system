# 编程作业智能批改系统

## 项目概述
基于Spring Boot和AI技术的编程作业智能批改系统，支持Java/Python/C++等主流教学语言的安全代码执行、智能分析和学习指导。

## 核心功能
- ✅ **安全沙箱执行**: Docker容器隔离，防止恶意代码
- 🤖 **AI智能分析**: GLM-4 API集成，提供详细错误解释和优化建议  
- 📚 **RAG知识增强**: 结合课程教材内容，提供针对性反馈
- 🎯 **多语言支持**: Java/Python/C++等主流教学语言
- 📊 **详细报告**: 代码质量评估 + 学习建议 + 知识点关联

## 技术栈
- **后端**: Spring Boot 3.x + MySQL + Docker + GLM-4 API
- **前端**: Vue3 (前后端分离)
- **AI**: 阿里云百炼GLM-4 + Chroma向量数据库
- **部署**: Nginx + Docker Compose
